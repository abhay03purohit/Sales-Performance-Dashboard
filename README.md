# Sales Performance Dashboard (Excel)

### Project Overview

This project is an interactive Sales Performance Dashboard built in Microsoft Excel. It provides data analysis, KPI tracking, and business intelligence reporting for multiple regions and sales executives. The dashboard enables users to monitor total sales, target achievement, and performance gaps through dynamic visualizations.

### Objectives
Track sales performance across regions
Measure Target Hit % and Away From Target %
Compare sales executives’ performance
Enable data-driven decision making
Provide clear visual insights using charts and KPIs

### Dataset Description

The dataset contains structured sales data with the following fields:

Emp Code – Unique employee identifier
Sales Executive – Employee name
Region – City/region assignment
Day1–Day5 – Daily sales records
Total Sales – Sum of weekly sales
Target – Assigned sales target
Target Hit % – Percentage of achieved target
Away From Target % – Remaining gap to target

### Dashboard Features
Region-based filtering (e.g., Mumbai, Delhi, Chennai, etc.)
Multiple dashboards for different KPIs
Visualizations:
Bar chart for Total Sales comparison
Pie chart for Target Hit distribution
Line chart for performance gap trends
Interactive slicers and controls
Clean and user-friendly layout

### Tools & Technologies
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Filters
Conditional Formatting
Data Aggregation Functions

### How to Use
Download the Excel file from this repository
Open in Microsoft Excel (recommended: Excel 2016 or later)
Use the region tabs or slicers to filter data
Analyze KPIs and charts for insights
Modify or extend the dataset as needed

### Key Insights
Identify top and low-performing sales executives
Monitor regional sales trends
Track target achievement vs shortfall
Support performance evaluation and strategy planning
